Countdown & Stopwatch App

A simple yet powerful application designed for timing events with precision.
 The app features a user-friendly interface allowing users to set countdown timers for any occasion and utilize a stopwatch for tracking
  elapsed time. Key functionalities include:

Countdown Timer: Set custom timers for events with alerts.
Stopwatch: Start, pause, and reset with a single tap.
Multiple Timers: Create and manage multiple countdowns simultaneously.

This app is ideal for anyone needing reliable timing tools for workouts, cooking, studying, or any timed activity.
